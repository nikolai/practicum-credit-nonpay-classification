# Модель предсказания дефолта по кредиту (поведенческий скоринг)
## Описание

Ключевые слова: классификация, RandomForest, calibration


## Требования

- **Python**: 3.11 или выше
- **ОС**: Linux, macOS или Windows (с WSL)

## Установка и запуск

#### Для Fish shell:

```fish
fish start.fish
```

#### Для Bash/Zsh:

```bash
./start.sh
```
